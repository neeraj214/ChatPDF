# ChatPDF

ChatPDF is a modern React-based web application that allows users to upload PDF documents and interact with them using an AI-powered chat interface.

## 🚀 Project Status

The frontend is currently **fully implemented** with the following pages and features:

- **Home Page**: Welcoming landing page with features overview and call-to-action.
- **Upload Page**: Interactive drag-and-drop zone for uploading PDF files.
- **Chat Page**: Clean, conversational interface for asking questions about your PDFs.
- **API Integration**: Ready-to-use service layer for connecting with a backend.
- **UI/UX**: Responsive design using Tailwind CSS, enhanced with Framer Motion and Lottie animations.

### Current Progress
- [x] Initial React + Vite setup
- [x] Frontend architecture (Routes, Layout, Components)
- [x] UI/UX implementation with dark mode support
- [x] PDF Upload interface with progress tracking
- [x] Chat interface with typing indicators
- [x] Git repository initialization and GitHub connection
- [ ] Backend API implementation (Currently using mock base URL `/api`)
- [ ] Production deployment

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS, PostCSS
- **Animations**: Framer Motion, Lottie-React
- **Icons**: React-icons
- **Notifications**: React-Toastify
- **Routing**: React Router DOM 7
- **API Communication**: Axios

## 📂 Folder Structure

- `src/components`: Reusable UI components (Uploader, Chat, Layout)
- `src/pages`: Application views (Home, Upload, Chat)
- `src/services`: API communication logic
- `src/assets`: Images and Lottie animations
- `public`: Static assets

## 🏁 Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/neeraj214/ChatPDF.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
